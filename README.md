# Form-Vue
A Form built with Vue.

### Features
+ Fully responsive
+ Beautiful design
+ Full form validation and error handling
+ Asynchronous email validation on the server
+ Animations

##### Built With
+ HTML5
+ SASS
+ Vue
+ Vuelidate

### Getting Started
clone repo

`cd` into folder and run `yarn install`

run `yarn serve`
